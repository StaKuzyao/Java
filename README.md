Для проверки использую Postman. Для отправки запроса выбираю метод GET. Для города прописываем URL: http://localhost:8080/weather/city В переменной city пишу город и получаю данные о погоде в этом городе. Например http://localhost:8080/weather/city?city=Полоцк Для координат прописываем URL: http://localhost:8080/api/weather/coordi Мы прописываем lat и lon и получаем данные о погоде в данных координатах, если сайт не определяет город, то прописывается null. Например http://localhost:8080/weather/coord?lat=55.7558&lon=37.6176
